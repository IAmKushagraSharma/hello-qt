# hello-qt

## Instructions to build the application

- First move in the root of the project

```bash
cd hello-qt
```

- Create build directory

```bash
mkdir build
```

- Move into the build dict

```bash
cd build
```

- Run cmake command

```bash
cmake ..
```

- Finally run the make command

```bash
make
```

That's it, now you can run the the binrary
```bash
./hello-qt
```
